📥 Comment utiliser ces icônes dans Power BI

PNG 64px → à utiliser dans le visuel Image pour les KPI cards et la navigation.
PNG 128px → pour le logo header ou les grands visuels dans un fond sombre.
SVG → compatible avec le visuel Image dans Power BI Desktop (File → Import → Image).
Conseil → Pour associer l'icône à une mesure DAX, héberge le PNG sur OneDrive et référence l'URL dans une colonne Image.
